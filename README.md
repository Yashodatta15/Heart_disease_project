# Heart disease project

## Objective

The objective of this project is to classify sonar signals as either rocks or mines using machine learning techniques. This enhances accuracy and efficiency in marine exploration and defense applications.

## Work

The project involves data preprocessing, feature selection, model training, and evaluation using various machine learning algorithms to achieve optimal classification performance.

## Tools

Python

Pandas

NumPy

Scikit-learn

Jupyter Notebook

## Conclusion

This project demonstrates the effectiveness of machine learning in identifying underwater objects using sonar signals. The developed model provides a robust solution for automating the classification process, offering potential improvements in safety and operational efficiency for marine applications.
